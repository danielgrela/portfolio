Este repositorio contiene el código fuente de mi portfolio personal. Es un proyecto moderno diseñado para ser rápido, accesible y fácil de mantener, aprovechando la potencia de Astro como framework estático y Tailwind CSS para el estilado.

> ⚠️ **Nota:** Este proyecto se encuentra actualmente en **desarrollo activo**. El diseño, secciones o funcionalidades pueden cambiar drásticamente.

## 🚀 Tecnologías

Este proyecto está construido principalmente con:

- **[Astro](https://astro.build/):** Framework web todo en uno para construir sitios web rápidos y centrados en el contenido.
- **[Tailwind CSS](https://tailwindcss.com/):** Framework de utilidades CSS para un diseño rápido y responsivo.

## 🛠️ Instalación y Uso

Si deseas ejecutar este proyecto localmente, sigue estos pasos:

1. **Clonar el repositorio:**

   ```bash
   git clone https://github.com/tu-usuario/nombre-repo.git
   cd nombre-repo
   ```

2. **Instalar dependencias:**

   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**

   ```bash
   npm run dev
   ```

   El sitio estará disponible en `http://localhost:4321`.

- `npm run dev`: Inicia el servidor de desarrollo local.
